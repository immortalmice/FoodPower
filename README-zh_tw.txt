=== A Minecraft mod about using power from foods. ===

食物會給予你力量......這早在你餓肚子的那天你就意識到了。
但吃著烤馬鈴薯、吃著烤牛肉、生啃胡蘿蔔的你，開始意識到這不太對。

食物可以變得更加有力量，超乎所有人的想像。

你開始嘗試把各種食材經過「 食譜設計 」，然後「 調理 」，變成全新的「 料理 」。
你發現到料理會激發出原先食材的力量，不同的食材會帶來不同的效果。
當然，這一路上並不是如此的順遂，你回想起把豬肉放進蛋糕的那一天......噁！

經過吃下自己所製作的料理，你獲得了強大的力量。
興高采烈的你，回過頭看自己之前所寫的食譜，發現有些地方並不太對。
於是你重新撰寫了一份新的食譜，不出意外，新的食譜比起舊的更加節省食材。

嗜甜的你，製作並吃下了大量的甜點，意外的，你發現你的身體開始變得輕盈了起來。
同時，偶爾血糖過高造成的致盲效果為你的生活帶來了不便。
你知道，這是食物的風味給你的額外力量，你還依稀可以在心中描繪出那六大風味。

但是，變更強大了，然後呢？

你發現好像有個天花板在那裡，阻擋你獲得更強大的力量。
你想起打從一開始製作料理，不小心打翻湯麵的那時，你發現到的三秒定理。
如果......大量的浪費食物......會不會可以有所進展？

你提起你的鑽石劍，準備了大量的甜點，你知道，這是一場艱難的戰役！
